# 命名约定

| 条目                      | 约定        |
| ------------------------- | ------------------ |
| Module                  | `大驼峰式`   |
| Types                     | `大驼峰式`   |
| Functions                 | `大驼峰式`   |
| Methods                   | `大驼峰式`   |
| Generic parameters        | `大驼峰式`   |
| Variables                 | `小下划线式` |
| Member variables          | `小下划线式` |
| Keywords                  | `小下划线式` |
| Type literals             | `小下划线式` |
| Boolean type and literals | `小下划线式` |

其他的一些补充

* 如果一个符号我们不建议外部直接操作它, 我们可以在前面加上"_", 如"_length"
* 如果一个符号是不安全的或者原始的, 我们则在前面加上"__", 如"__cast"
